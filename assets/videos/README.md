# 视频资源说明

本目录用于存放张家界国家森林公园官网的宣传视频。

## 所需文件

请将宣传视频文件命名为 `promotional.mp4` 并放入此目录。

## 视频规格要求

| 项目 | 推荐规格 |
|------|----------|
| 文件名 | `promotional.mp4` |
| 格式 | MP4 (H.264 编码) |
| 分辨率 | 1920×1080 (1080p) |
| 时长 | 30秒 - 3分钟 |
| 文件大小 | 不超过 100MB |

## 视频内容建议

- 展示张家界国家森林公园的核心景点（袁家界、金鞭溪、天子山等）
- 包含航拍镜头展示石柱群
- 展示季节特色（春花、夏云海、秋色、冬雪景）
- 背景音乐建议使用轻音乐或自然音效

## 获取视频素材

可以从以下途径获取免费/付费视频素材：
- Pexels Videos: https://www.pexels.com/zh-cn/video/
- Pixabay Videos: https://pixabay.com/zh/videos/
- Videvo: https://www.videvo.net/
- Pond5: https://www.pond5.com/

或联系专业视频制作团队拍摄定制。

## 注意事项

1. 确保视频版权合规，避免侵权问题
2. 上传后请在浏览器中测试视频播放效果
3. 如果视频无法播放，可能是编码格式问题，请使用 HandBrake 等工具重新编码
